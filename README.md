# console.md

The missing console.md(msg) to print markdown with the nodejs console object

# Install

    npm i maboiteaspam/console.md

# Usage

```js
    require('console.md')();

    console.md('__hello__ --the world-- !');

    console.mdLine('__hello__ --the world-- !');

```
